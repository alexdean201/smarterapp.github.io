Implementation Readiness

The resources on this page are what the Smarter Balanced team have been identified as necessary to implement Smarter Balanced Assessments are what comprise the Implementation Readiness Package (IRP).

The purpose of the Implementation Readiness Package is to connect the technical assessment implementation teams of members and service providers with the resources that are necessary to administer a Smarter Balanced assessment. Please use the Overview Document as a tool in preparing for and maintaining Smarter Balanced Assessments.
